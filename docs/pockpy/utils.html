utils¶

Module containing various auxiliary functions used in other modules.

pockpy.utils.compute_available_closed_orbit(aperture_scan_df, limit, col='N1')¶

Computes available closed orbit on input aperture scan table and limit.

Parameters

  • aperture_scan_df (pd.DataFrame) – DataFrame from a call to core_io.aperture_scan().

  • limit (pd.Series) – Series sharing the index of py:data:`aperture_scan where each entry specifies the minimum acceptable aperture of the kind specified by py:data:`col.

  • col (str) – String specifying which of the aperture outputs is constrained. Should be one of ‘N1’, ‘N1X_M’ or ‘N1Y_M’.

Returns

Series containing the maximal closed orbit uncertainty for which the aperture limit is satistfied per element.

pockpy.utils.pinv(A, n)¶

Performs a pseudoinverse on a numpy.2darray or pandas.DataFrame for a given number of singular values.

Parameters

  • A – DataFrame or numpy.2darray to be pseudoinverted.

  • n (int) – The number of singular values to used for the pseudoinverse.

Returns

Pseudoinverse of A for the given number of singular values. If A is a DataFrame, the pseudoinverse will be a DataFrame with index and columns swapped around.

Raises

ValueError – If invalid n is provided.

pockpy.utils.pinv_per_dim(df, n_x, n_y=None)¶

Performs the pseudoinverse on a response matrix independently in each dimension.

Parameters

  • df (pd.DataFrame) – DataFrame to be pseudoinverted.

  • n_x (int) – Integer representing the number of singular values to use for the pseudo-inverse in the horizontal plane.

  • n_y (int) – Integer representing the number of singular values to use for the pseudoinverse in the vertical plane. Set to n_x if not provided.

Returns

Pseudoinverse of df (applied per dimension) expressed as a DataFrame with index and columns swapped around.

pockpy.utils.remove_duplicates(itr)¶

Removes duplicate elements from an iterable while retaining order and returns result as a list.

The order of the input is maintained in the sense that the first occurrence of any given element defines its order.

Example:

>>> li = ['a', 'b', 'c', 'b' 'a']
>>> remove_duplicates(li)
['a', 'b', 'c']
>>> tup = ('b', 'a', 'b')
>>> remove_duplicates(tup)
['b', 'a']
Parameters

itr – An iterator.

Returns

A list with all duplicate elements removed and with intact ordering.
